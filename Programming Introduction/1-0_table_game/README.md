# 1 and 0 game

<div align="center"> 
In this game, you need to turn all 1 numbers to 0!
 </div>
 <div align="center"> 
 When you change one number, all of the side numbers change too!
 </div>
 <div align="center"> 
 You need to think of a strategy to turn the entire board to 0 and win the game! Good luck!
  </div>
